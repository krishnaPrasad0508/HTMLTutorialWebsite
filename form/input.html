Input

The HTML <input> tag is used to represent a form input control in HTML document. This form input control facilitate user to input data and communicate with a website or application.

See this example:

 <form action="#">  
    First name: <input type="text" name="FirstName"  placeholder="enter firstname..."><br>  
    Last name: <input type="text" name="LastName" placeholder="enter lastname..."><br>  
    <input type="submit" value="Submit">  
</form>

Output:

First name:
Last name:
